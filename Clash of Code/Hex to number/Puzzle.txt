Shortest mode

Find the ascii value of the hex input n. The ascii spells out the number you need to find. e.g. n is "7468726565" which translates to ascii string: "three", therefore print 3.

Input
A string n which is a sequence of pairs of hex values
Each pair is a representation of an ascii character

Output
A number corresponding to the ascii value of the input,

Constraints
output is a number between 0 and 9, both included

Example:
Input:
4F4E45

Output:
1
