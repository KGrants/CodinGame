You must determine whether or not the first word is an encrypted version of the second word by using a Caesar letter substitution cipher, in which each letter is replaced by the letter a fixed number of places later or earlier in the alphabet.

Input
Line 1: Uppercase letter sequence A.
Line 2: Uppercase unencrypted word B.

Output
Line 1: Either true or false.

Constraints
len(A) = len(B)
2 < len(A), len(B) < 20

Example
Input
TUSPOH
STRONG

Output
true
